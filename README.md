# Healthcare Data Analysis System

A data analysis project using MySQL and Python
to analyze patient and treatment data.

## Technologies Used
- MySQL (Database & Stored Procedures)
- Python (pandas, matplotlib, mysql-connector)
- Excel (Automated Reports)

## Features
- Patient database with 20 records
- 10+ SQL analysis queries
- 2 stored procedures
- Automated Excel report generation
- Data visualizations (charts)

## How to Run
1. Run create_tables.sql in MySQL
2. Run insert_data.sql to load data
3. Run queries.sql to see analysis
4. Run stored_procedures.sql
5. Run: python analysis.py

## Output
- healthcare_report.xlsx
- patients_by_department.png
- doctor_revenue.png
